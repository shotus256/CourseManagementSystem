# Система управления курсами повышения квалификации

![Логотип приложения](logo.png)

**Курсовой проект** по дисциплине «Проектирование и администрирование баз данных»  
Разработка клиент-серверного desktop-приложения для организации курсов по повышению квалификации.

## Описание проекта

Информационная система предназначена для автоматизации управления курсами повышения квалификации в учебном заведении. Приложение позволяет:

- **Методисту:** управлять преподавателями, нагрузкой, курсами, группами, дисциплинами, специальностями и пользователями.
- **Преподавателю:** просматривать свою нагрузку, ставки и общую оплату.

## Технологии

- **Язык программирования:** C++17
- **Фреймворк:** Qt 6 (GUI, SQL)
- **СУБД:** PostgreSQL
- **Архитектура:** MVC (Model-View-Controller)
- **Система контроля версий:** Git

## Основные функции

### Для методиста:
- Управление преподавателями (CRUD)
- Управление нагрузкой
- Управление курсами
- Управление группами
- Управление дисциплинами
- Управление специальностями
- Управление пользователями и ролями

### Для преподавателя:
- Просмотр личной информации
- Просмотр своей нагрузки
- Просмотр ставок и общей оплаты

## Установка и запуск

### Требования
- PostgreSQL 14 или выше
- Qt 6.0 или выше (для сборки из исходников)

### Инструкция
1. **Клонируйте репозиторий:**
```bash
git clone https://github.com/shotus256/CourseManagementSystem.git
cd CourseManagementSystem
```
2. **Создайте базу данных:**
```sql
CREATE DATABASE training_courses;
```
3. **Выполните SQL-скрипт для создания таблиц (файл init_db.sql):**
```bash
psql -d training_courses -f init_db.sql
```
4. **Настройте подключение к БД в файле Database/DatabaseManager.cpp:**
```bash
bool DatabaseManager::connect(...) {
    m_db.setHostName("localhost");
    m_db.setPort(5432);
    m_db.setDatabaseName("training_courses");
    m_db.setUserName("ваш_пользователь");
    m_db.setPassword("ваш_пароль");
    // ...
}
```
5. **Откройте проект в Qt Creator и выполните сборку**
---
Для macOS собранный .app файл доступен в релизах репозитория или может быть пересобран из исходников с помощью Qt.
